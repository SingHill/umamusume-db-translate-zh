ウマ娘数据库翻译补丁

本项目通过修改 master.mdb 中的 text_data 表来实现 ウマ娘 プリティーダービー 的客户端翻译。
修改过程可完全通过该网页完成。如果你有兴趣帮助该项目，请前往 GitHub 储存库。

使用方法

  1. 找到你的 master.mdb 文件
    2. 它通常存放在 C:\Users\[你的用户名]\AppData\LocalLow\Cygames\umamusume\master\master.mdb
  2. 使用此网页已翻译的 master.mdb 文件替换游戏原本的 master.mdb 文件
  3. 请一定要备份 master.mdb 文件，并使用游戏原本的 master.mdb 文件在本页进行替换。如果你不小心弄丢了你的原始文件，我们会在 Release 中发布最新的原始版本和最新汉化版本

注意事项

  • 修改游戏文件违反了游戏的服务条款！
    使用该工具的后果请自负！我们不会对你的账号负责！
  • 不是所有的字符串都包含在 master.mdb 当中，所以此方法不能完全汉化客户端。
  • 由于有字符串长度上限，所以部分技能描述看起来会很奇怪。
  • 如果游戏更新，你需要重新替换 master.mdb文件。
  • 这个工具主要是为 DMM 版本的 PC 端而制作的，因为 PC 端替换文件的方式比较简单。
    • 你同样可以在移动设备上通过替换 master.mdb 文件来完成汉化，只不过比较麻烦。
